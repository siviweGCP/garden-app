# Garden App

Garden App is a simple Python application that gives gardening advice based on the month and season. It is designed to help gardening enthusiasts decide what to plant and how to care for their garden throughout the year.

## Features

- Provides monthly planting advice.
- Provides seasonal gardening tips.
- Uses reusable functions to keep the code organised.
- Demonstrates basic Python programming concepts.
- Shows a professional GitHub workflow using issues, branches, commits, pull requests, and merges.
